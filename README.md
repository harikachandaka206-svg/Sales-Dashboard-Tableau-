# 📊 Sales Dashboard using Tableau

## 📌 Project Overview

This project presents an interactive Sales Dashboard developed using Tableau to analyze sales performance and business trends. The dashboard enables users to monitor key performance indicators (KPIs), compare sales across different regions and categories, and identify valuable business insights through interactive visualizations.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-performing regions and product categories.
- Track monthly sales trends.
- Provide interactive filtering for better decision-making.
- Present business insights through visual dashboards.

---

## 🛠️ Technologies Used

- Tableau
- Microsoft Excel / CSV Dataset
- Data Visualization

---

## 📈 Dashboard Features

- Total Sales KPI
- Profit Analysis
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Interactive Filters
- Dynamic Charts

---

## 💡 Key Insights

- Identified top-performing regions based on sales.
- Compared sales across different product categories.
- Analyzed monthly sales growth.
- Highlighted important business trends using interactive dashboards.
